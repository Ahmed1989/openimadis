iManage ImageJ Plugins

Release Date: November 24, 2015

Set-Up

Download

iManage ImageJ Plugins zip can be downloaded from here

Install

Unzip the file inside plugins folder in ImageJ and restart it. You will see "iManage" in plugins menu. Please make sure that ImageJ is working with Java 1.7. These plugins will not work with java 1.6.

Plugin 1: View Record

  • Generate the auth-code using "Settings > Manage Auth-Tokens" menu in iManage webclient.
  • Generate auth-codes for imagejclient_sample client.
  • Run the plugin and provide auth-code.
  • This will open the record matching search criterion in ImageJ.

Plugin 2: Edge

This plugin invokes a processing in ImageJ and uploads modified image back to iManage.
