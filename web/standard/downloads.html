Acquisition Client

Acquisition Client - Download

iManage Acquistion Client can be downloaded from here.

Acquisition Client - Install

  • Extract iManage Acquistion Client to a local folder
  • Run the script runAcq.sh or runAcq.bat. You need java 1.7+ to use acquisition client.
  • If you are connecting to a server on https, use additional instructions in README to install the server certificate.
Here are some details on functionality supported by the acquisition client.

API - Example

iManage API Java example can be downloaded from here.
It can be compiled with client-api.jar present in the zip downloaded from above link. client-id and auth-code needed bt the application need to be generated using the webclient. Current clientid is a valid id for a testclient. Authcode for this client can be generated by the user using Settings menu in webclient.
