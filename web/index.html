openImaDis - Image Management System for Microscopy

Strand's openImaDis is an advanced software solution for the images generated by automated microscopy. It has a image management system for visualisation and organisation of large, complex multi-dimensional data sets. The server supports a java based Acquisition Client to upload images in original acquired format. A browser based web client allows visualisation, annotations, sharing and download of images by users based on their permission. A java API is provided to directly access data from third party applications.
